# 💡 БУРМОЛДА — что добавлено и что можно добавить дальше

Отчёт по прокачке боёвки/сложности + бэклог идей для развития игры.
Оценки: 🟢 быстро (вечер) · 🟡 средне (пара вечеров) · 🔴 крупно (неделя+).

---

## Часть 1. Что только что добавлено (для контекста)

### Боссы и хард стали настоящими
- **HP боссов**: множители подняты — Калитин ×1.9, Цизи ×1.6, Жижа ×1.7
  (`data/balance/combat.json → BOSS_HP_MULT`), плюс скейл от уровня игрока ×6
  (было ×2). Калитин теперь ~215 HP вместо ~90 — бой на 8–12 ходов.
- **Хард-мобы**: HP +9% и урон +5% за каждый пункт опасности локации выше 2
  (`battle.gd`). На опасности 9 моб в ~1.6 раза толще, чем тот же на 1.
- **Награды за хард**: бурмолда `+danger*6`, кринж-опыт `+danger*3` — качаться
  в опасных локациях теперь ВЫГОДНО, а не просто больно.
- **Лут стал сильнее**: тир дропа `danger/2` (раньше /3), появился НИЖНИЙ порог
  тира (в харде мусор не сыпется), с боссов снаряжение падает **гарантированно**
  и на тир выше (`loot.gd`).

### Боевая система — паттерны уворота (в духе Undertale)
Вместо одних шариков — 7 типов атак (`battle_scene.gd`):

| Паттерн | Что происходит | У кого |
|---|---|---|
| РОЙ (aimed) | снаряды с краёв летят в душу | все |
| ЛИВЕНЬ (rain) | капли сверху дождём | все; Калитин, Жижа |
| КОСТИ (walls) | стены костей слева/направо с ПРОХОДОМ — ищи щель | опасность 3+; Калитин |
| ВИХРЬ (spiral) | спираль снарядов из центра | опасность 3+; Цизи, Жижа |
| ПРЕСЛЕДОВАТЕЛИ (homing) | крупные орбы доворачивают к душе | опасность 6+; Жижа |
| ЛУЧИ (laser) | полоса-телеграф мигает → бьёт лучом | опасность 6+; Калитин |
| ШКВАЛ ЦИЗИ (wind) | ветер физически СНОСИТ душу + снаряды | только Цизи |

- У **боссов фирменные наборы**: Калитин — кости/лучи/ливень (скелет-барабанщик),
  Цизи — ветер/вихрь (дух сквозняка), Жижа — ливень/преследователи.
- У обычных мобов пул паттернов **растёт с опасностью локации** — на болоте у
  дома только рой и ливень, в бездне — всё сразу.
- Длительность фазы уворота растёт с опасностью, у боссов дольше.
- Название атаки показывается («⚠ КОСТИ!») — читаемость как в Undertale.

## Часть 2. Идеи — боевая система

- ✅ ~~Цвет = опасность~~ — СДЕЛАНО упрощённо: 🟢 зелёные снаряды безвредны
  (обманки), остальные бьют. Полный Undertale-вариант (синий не бьёт стоящего,
  оранжевый — движущегося) остаётся идеей на потом: флаг у снаряда + проверка
  скорости души.
- 🟡 **Режимы души**: синяя душа (гравитация — падаешь, прыжок), фиолетовая
  (движение по «строкам»). Для боссов вторая фаза = смена режима души.
- ✅ ~~Фазы боссов~~ — СДЕЛАНО: на 50% HP босс свирепеет (+урон, шире пул атак,
  быстрее спавн, анонс). Плюс сделаны: комбо из 2 паттернов, цвет пули = скорость,
  уворот от Блок%/зелья, тряска экрана, «дышащая» коробка.
- 🟢 **Пре-телеграф направления костей**: восклицательный знак у стены, откуда
  пойдут кости (честность в харде).
- ✅ ~~Атаки со «спецэффектом от биома»~~ — СДЕЛАНО: лёд = инерция души,
  вулкан = раскалённые стены коробки, болото/вода = вязкость. Плюс пиксель-арт
  плакаты всех боссов (утверждены).
- 🟡 **Милость/пощада как в Undertale**: у части мобов появляется условие
  «пощадить» (после N ходов ACT) → бой без убийства, награда репутацией.
  ACT-опции на моба: «подразнить», «сделать комплимент», «бурмольнуть тихо».
- 🟡 **Щит-парирование**: с щитом в фазе уворота по нажатию Space — блок 0.3с
  (гасит 1 снаряд, кулдаун); block% из статов наконец заиграет.
- 🔴 **Уникальные бои-головоломки**: у финальной Жижи коробка меняет форму,
  у Калитина — «синие кости» в такт барабанам (ритм-атака).

## Часть 3. Идеи — прокачка и глубина

- 🟢 **Очки характеристик**: на левелапе +2 очка, распределяешь в АТК/ЗАЩ/HP/УДАЧУ
  (удача = шанс лута/крита). Даёт билды: «танк», «крит-убийца», «фармер».
- 🟡 **Таланты по рангам**: каждый ранг (Болотный Новичок, Лиановый Сигма…)
  даёт пассивку на выбор из двух: «+1 клетка скольжения» vs «иммунитет к траве»,
  «вампиризм +5%» vs «шипы +2» и т.п.
- 🟡 **Крафт у кузнеца**: ресурсы + бурмолда → предмет тир+1 из двух одинаковых
  («перековка»). Замыкает экономику: добыча → продажа → покупка → перековка.
- 🟢 **Сет-бонусы**: полный «Свэговый» комплект (меч+броня+щит одного тира) даёт
  бонус («Свэг-аура»: +10% крит). Данные — просто словарь тир→бонус.
- 🟡 **Зачарование колец**: слить два кольца одного эффекта → +1 ступень.
- 🟢 **Эликсиры из трав**: собранные травы можно не продавать, а варить у
  Шестухины: зелье защиты (на 1 бой), зелье удачи (лут x2 на 3 боя).

## Часть 4. Идеи — мир и контент

- 🟢 **Мини-карта** (M): схема локаций-узлов с отметкой «ты здесь», открывается
  по мере исследования.
- 🟡 **Погода/время**: дождь на болоте (мобы злее, травы больше), ночь в лесу
  (спавн редких «ночных» мобов с жирным лутом).
- 🟡 **Новые боссы** (данные уже позволяют): Гриб-Патриарх в джунглях (споры-
  преследователи), Ледяной Тенор-EX на опере (лучи по ритму), Костяная Мать
  в катакомбах (стены костей с ДВУМЯ проходами, один ложный).
- 🟢 **Секретки**: 3–5 скрытых клеток-тайников на картах (декор-подсказка
  рядом), внутри — кольца/рецепты. Проверка клетки уже есть (ноды).
- 🟡 **НГ+ (Новая игра+)**: после победы над Калитиным — рестарт с сохранённой
  экипировкой, мобы ×2 HP, новый банк реплик «болото помнит тебя».
- ✅ ~~Бесконечное подземелье~~ — СДЕЛАНО в Адской Шахте: бесконечные этажи,
  лава/озёра с рыбами-стрелками, мини-боссы (5) / боссы (10) / ТМ (25), лор Пекла.
  Осталось: таблица рекордов глубины (пара строк в сейве + экран).
- 🟢 **Рыбалка-полноценная**: пойманную «золотую рыбину свэга» можно отнести
  Злой Рыбе — она либо съест (репутация), либо оскорбится (бой).

## Часть 5. Идеи — интерфейс и звук

- 🟢 **Тряска экрана** при уроне/крите (2–3 строки: смещение камеры на randf).
- 🟢 **Всплывающие цифры урона** над врагом (Label с твином вверх+fade).
- 🟡 **Музыка**: процедурный чиптюн-луп для надмира/боя/босса (AudioStreamWAV
  генерится так же, как SFX — арпеджио из 4 нот по биому).
- 🟢 **Вибрация меню** при отказе (нет денег и т.п.) — сдвиг меню на 2px туда-сюда.
- 🟡 **Портреты эмоций**: у NPC 2-3 варианта спрайта (злой/довольный) — сетки
  уже есть, нужны копии с другим ртом/глазами.

## Часть 6. Технический бэклог

- 🟡 **Фаза 5 — экспорт**: Windows .exe + Web-сборка (одна кнопка, но нужно
  добавить `*.json` в фильтры экспорта!). Отложено по решению владельца.
- 🟢 **Сейв-слоты** (3 штуки + автосейв при переходе локаций).
- 🟡 **Настройки**: громкость, скорость текста, размер окна, ремап клавиш.
- 🔴 **Мультиплеер-заготовка**: у Godot встроенный high-level netcode; в старом
  проекте есть протокол (`burmolda-python/core/net.py`) — co-op на 2 головастиков.
- 🟢 **Больше тестов**: симулятор 100 боёв на каждом уровне → таблица TTK/смертей
  (авто-проверка кривой сложности при изменении баланса).

### С чего советую начать (максимум эффекта за вечер)
1. Синие/оранжевые снаряды — глубина уворота почти бесплатно.
2. Тряска экрана + цифры урона — бой сразу «сочнее».
3. Очки характеристик — прокачка обретает выбор.
4. Сет-бонусы + перековка — экономике появляется цель.
